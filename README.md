# SmartCart – Customer Segmentation System

## Overview

SmartCart is a Machine Learning project that analyzes e-commerce customer data and groups customers into meaningful segments. The system helps businesses understand purchasing behavior and design targeted marketing strategies.

## Key Focus

The main focus of this project is **data preprocessing and exploratory data analysis (EDA)**, which are critical steps in preparing raw data for machine learning models.

## Methodology

* Cleaned and prepared the dataset
* Performed **Exploratory Data Analysis (EDA)** to identify patterns
* Applied the **Elbow Method** to determine the optimal number of clusters
* Implemented **K-Means Clustering** to segment customers

## Tech Stack

Python, Pandas, NumPy, Matplotlib, Seaborn, Scikit-learn, Jupyter Notebook

## Outcome

Customers were grouped into segments based on **income and spending behavior**, helping businesses identify high-value customers and improve marketing strategies.
